The python version for cobot
############################

Download
========

.. code-block:: bash
    git clone https://github.com/victoryang/cobot-python

Prequisite
==========

.. code-block:: bash
    pip

Setup tools
===========

.. code-block:: bash
     pip install setuptools
     python setup.py sdist

Install
+++++++

.. code-block:: bash
     pip install ./dist/cobot-python-0.1.tar.gz
